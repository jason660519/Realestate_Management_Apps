## 完整流程

### 步驟一：掃描今日工作內容

- 回顧本次對話中所有已完成的任務
- 彙整每項任務的交付物與完成度

### 步驟二：撰寫 Dev Log

存到 `docs/project-process/dev-logs/dev-{topic}-{YYYY-MM-DD}.md`：

1. **本日完成任務清單**（條列式，含交付物與完成度）
2. **技術困難**（問題 → 排查 → 根因 → 解法）
3. **踩雷事件**（重工/延遲情境 + 事前可預防指標）
4. **下次避免措施**
5. **明日優先工作**（預估工時、相依性、風險）

### 步驟三：輸出摘要

```
✅ 已完成每日進度報告

📄 Dev Log: docs/project-process/dev-logs/dev-{topic}-{YYYY-MM-DD}.md
```
